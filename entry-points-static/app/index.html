Worspace Examples!- by PoliteJS

Monolithic Single Page Application

This examples uses KnockoutJS' Components to split the app into sub-modules and a router to drive the correct module to load. It does support deep linking!

Everything is bundled into one single file which is not a really optimal choiche if you care about page loading performances!

Please take a look at the Dynamic Entry Points example for a different and more optimal implementation.
