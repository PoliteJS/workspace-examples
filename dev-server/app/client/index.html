Dev Server Integration

This app loads a list of colors from a JSON end point exposed within the development server:

  • http://localhost:8080/colors
  • http://localhost:8080/colors/1

We play a little bit with the workspace.conf.js in order to split out /app folder into /app/client and /app/server then we can set a js file to be included and executed within the develompend server.

The server entry point is part of an ExpressJS application!
