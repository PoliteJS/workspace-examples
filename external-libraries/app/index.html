Requiring External Libraries

This examples uses jQuery and Twitter Bootstrap from cdnjs.cloudfare.com.

Using CDN is a really good strategy to improve page loading performances!

Then I want to use jQuery from my application entry point which is a CommonJS module, I don't want to access the global scope with window.$, instead I do prefer the more secure and standard way to explicitly require all the module's dependencies with var $ = require('jquery').

Take a look at webpack.config.js for the configuration!
